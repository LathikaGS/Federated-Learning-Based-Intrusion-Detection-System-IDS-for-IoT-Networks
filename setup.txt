# NS-3 Setup on Windows using WSL

This guide helps you set up NS-3 on a Windows system using WSL (Windows Subsystem for Linux). It is intended for running DDoS simulations (`ddos_ns3.cc`) and processing results using a Federated Learning LSTM model (`fl_ddos.py`).

---

## Step 1: Install WSL

Open **PowerShell as Administrator** and run:

```powershell
wsl --install
````

Restart your system when prompted. This installs Ubuntu as the default WSL distribution.

---

## Step 2: Open WSL Terminal

Search and open **"Ubuntu"** from your Start Menu.

---

## Step 3: Install NS-3 Dependencies

Run the following commands in the Ubuntu WSL terminal:

```bash
sudo apt update
sudo apt install -y g++ python3-pip qt5-default mercurial cmake \
libsqlite3-dev libgtk-3-dev libboost-all-dev \
libxml2-dev libgsl-dev libgmp-dev \
python3-setuptools python3-pygraphviz graphviz \
gir1.2-goocanvas-2.0 doxygen imagemagick
```

---

## Step 4: Install Python Dependencies

Make sure you have your `requirements.txt` file ready. Then run:

```bash
pip3 install -r requirements.txt
```

> This installs dependencies needed for running `fl_ddos.py`.

---

## Step 5: Download and Build NS-3

Clone the NS-3 repository and build it:

```bash
git clone https://gitlab.com/nsnam/ns-3-dev.git
cd ns-3-dev
./ns3 configure --enable-examples --enable-tests --enable-modules=netanim
./ns3 build
```

---

## Step 6: Place and Run Simulation Code

1. Place your simulation file `ddos_ns3.cc` in the `scratch/` directory.
2. Run the simulation and the federated learning script:

```bash
./ns3 run scratch/ddos_ns3
python3 scratch/fl_ddos.py scratch/ddos_attack_data.txt
```

---

## Step 7: View NetAnim Output

If you want to visualize the simulation GUI, install an X server like **VcXsrv** on Windows. Then launch the animation:

```bash
netanim DDoS_Scattered.xml
```

---

## Notes

* The `.csv` file generated by `ddos_ns3.cc` is used as input for `fl_ddos.py`.
* Ensure that all file paths (e.g., `scratch/ddos_attack_data.txt`) match exactly.
